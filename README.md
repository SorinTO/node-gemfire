pivotal-gemfire
===============

Proof-of-concept node module connecting to Pivotal Gemfire

## Prerequisites 

* node 0.10.x http://nodejs.org/
* vagrant 1.6.x http://www.vagrantup.com/

## Setup
On your Mac

    $ vagrant up
    $ vagrant ssh

On the vagrant box

    $ cd /vagrant
    $ ./setup_ubuntu.sh

## Rebuild and run tests

    $ grunt
